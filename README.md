# Webcomic (Single-Page) — GitHub Pages

This is a static, single-page webcomic site.

## Add your comics
1. Put images in `assets/img/comics/` (example names: `001.jpg`, `002.jpg`, ...)
2. Edit `comics.json` to match filenames + titles + dates.

## Host on GitHub Pages
GitHub repo → Settings → Pages → Deploy from branch → `main` / root.

Open:
- `https://YOUR_USERNAME.github.io/REPO_NAME/`
